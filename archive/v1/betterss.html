Projeler Sayfasına Dön

BetterSS, bir ekran görüntüsü alma wrapperidir.

Platform desteği: Debian GNU/Linux, Fedora Linux, Pisi GNU/Linux tabanları
Arayüz: Tkinter & Shell
Lisans: GNU General Public License 3[GNU Genel Kamu Lisansı 3(GPL 3)]

BetterSS kurulumu

BetterSS'u Apiutaller kuracaktır.
Burada BetterSS için ayarlı olan Apiutaller'ı çalıştırma talimatları yer almaktadır.

1. yol: Terminalden çalıştırma:
  1. Eğer Debian GNU/Linux tabanlı GNU/Linux dağıtımı kullanıyorsanız terminale şu komutu yazın:
    sudo apt install git && git clone https://github.com/mukonqi/BetterSS.git && cd BetterSS && sudo python3 apiutaller.py
    2. Eğer Fedora Linux tabanlı GNU/Linux dağıtımı kullanıyorsanız terminale şu komutu yazın:
    sudo dnf install git && git clone https://github.com/mukonqi/BetterSS.git && cd BetterSS && sudo python3 apiutaller.py
    Eğer Pisi GNU/Linux tabanlı GNU/Linux dağıtımı kullanıyorsanız terminale şu komutu yazın:
    sudo pisi it git && git clone https://github.com/mukonqi/BetterSS.git && cd BetterSS && sudo python3 apiutaller.py
Artık Apiutaller açılacak ve bilgisayarınıza sizin yönlerdirmenize bağlı olarak BetterSS'ı kuracaktır.
2. yol: Elle çalıştırma:
  1. apiutaller.py adında bir dosya oluşturun.
  2. apiutaller.py dosyasını açın
  3. BetterSS bağlantısındaki kodları apiutaller.py dosyasına yapıştırın.
  4. apiutaller.py dosyasını çalıştırın.
Artık Apiutaller açılacak ve bilgisayarınıza sizin yönlerdirmenize bağlı olarak BetterSS'ı kuracaktır.

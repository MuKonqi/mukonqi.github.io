MetterXP, gelişmiş araç kutusu.

  1. Uygulama mağazası
  2. Masaüstü Ortamı ve Pencere Yöneticisi mağazası
  3. Paket yöneticisi mağazası
  4. Programları yapılandırma
  5. .bashrc dosyası yapılandırma
  6. Dosya yöneticilerini kök kullanıcı haklarıyla açma
  7. Bazı GNU/Linux dağıtımları hakkında bilgi verme
  8. Çeşitli yapılandırmalar (.bashrc, GRUB, Cups, Wine ve bilgisayar adı)
  9. Ve dahası...

Platform desteği: Debian GNU/Linux, Fedora Linux, Solus tabanları
Arayüz: Tkinter
Lisans: GNU General Public License 3[GNU Genel Kamu Lisansı 3(GPL 3)]

MetterXP kurulumu

  1. İlk olarak Apiutaller sihirbazı için bağımlılıkları yükleyeceğiz.
    • Eğer Debian GNU/Linux kullanıyorsanız ya da bu dağıtımları temel alıyorsanız konsola şunu yazın: sudo apt install git python3 -y
    • Eğer Fedora Linux veya onun tabanlı dağıtımlarını kullanıyorsanız konsola şunu yazın: sudo dnf install git python3 -y
    • Eğer Solus kullanıyorsanız ya da Solus dağıtımlarını temel alıyorsanız bunu konsola yazın: sudo eopkg install git python3 -y
  2. Şimdi depoyu indirmeli ve Apiutaller sihirbazını açmalıyız.
    git clone https://github.com/MuKonqi/metterxp ; cd metterxp ; sudo python3 apiutaller.py --install ; cd .. ; rm -rf metterxp
  3. İşte bu kadar!.

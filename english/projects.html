Apiutaller

Both BetterXP and MetterXP had their own installers (not wizard), but I wanted to make a global installer wizard, so I released the Apiutaller wizard. Apiutaller also made my own work easier in many of my subsequent projects.
With v2.0, Apiutaller got rid of childish and unnecessary sympathetic expressions. Also reinstallation feature added. For all the changes: Click here.

Muktart

What Muktart did: Run ls command for looking /usr/share/applications directory's items later lists the applications and launches them on user's demand (normally or with sudo). By the way, people start applications like this before start menu was invented (!).

MetterXP (stopped)

My former "big" project, but I stopped it because the interface and the code were too complicated, the code was not coded well at all, so I started a new project from scratch: GrelinTB

BetterXP (stopped)

I started to build BetterXP based on Terminalden kurtulun. This was both out of curiosity and to remove the logo from the home page of the Terminalden kurtulun project. The first BetterXP releases contained my own improvements. But in the later versions under 1.x I added new support to those improvements, namely Fedora Linux and Solus support. One of the reasons for adding support was that I was a distrohopper at the time.
After that, 2.0 came out and that was short-lived. I changed the colors there and I made some improvements for GUI. And finally it died because the code was very, very complicated.
